# WatchWorth

WatchWorth is a movie review platform where you can explore curated picks, see weekly recommendations, and manage the catalog through an admin panel.

## Features
- Home page with featured movies
- Movie collection page
- Admin dashboard to add/remove movies
- Auth-protected routes
- Responsive UI with dark theme

## Tech Stack
- React + TypeScript (Vite)
- React Router
- Tailwind CSS
- React Hook Form + Zod
